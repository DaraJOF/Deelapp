# Project Title

Cypress automation for Deel application.

## Description

Wrote a series of automated tests for creating a fixed contract on deel app.

## Getting Started

### Dependencies

* Cypress: ^9.2.0


### Executing program

* npm install
* npx cypress run --spec "cypress/integration/deelapp.js" --browser chrome --headed
